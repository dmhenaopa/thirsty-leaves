# 🌿 thirsty-leaves
> *Exploratory analysis of plant physiological responses to water stress.*

A data analysis project exploring how water stress affects key physiological variables in plants like stomatal conductance, water potential, photosynthesis — identifying patterns, correlations, and communicating findings through clear visualizations.

## 🎯 Objectives
1. Explore how water stress affects key physiological variables in plants
2. Identify patterns and correlations between stress indicators
3. Communicate findings in a visual and reproducible way
